# 社区行为准则

## 我们的承诺

为了建立一个开放和欢迎的社区，我们承诺：

- 使用热情和尊重对待所有参与者，无论其背景、性别、性取向、能力、种族、民族或宗教信仰
- 欢迎新手和学习者
- 专注于建设性和有益的讨论
- 尊重不同的观点和经验

## 我们的标准

### ✅ 我们期望的行为

- 使用欢迎和包容的语言
- 对不同的观点和经验保持尊重
- 优雅地接受建设性批评
- 专注于对社区最有利的事情
- 在得罪他人时表示同情

### ❌ 不可接受的行为

- 辱骂、贬低或骚扰行为
- 任何形式的性骚扰或性别歧视
- 仇恨言论或人身攻击
- 发布他人私密信息（"doxing"）
- 其他被合理认为不专业或不受欢迎的行为

## 执法

### 报告问题

如果你目睹或经历了不可接受的行为，请通过以下方式报告：

- 通过 GitHub Issues 中的私密讨论
- 直接 DM 项目维护者

**保密性**：我们承诺对所有报告者保密，并谨慎处理投诉。

### 处理流程

1. 收到报告后 48 小时内回复
2. 进行调查和事实核查
3. 确定适当的措施（警告、移除评论、封禁等）
4. 通知报告者和当事人结果

### 措施

- **首次警告** — 发送私密警告
- **评论删除** — 删除违反准则的评论
- **临时封禁** — 7 天无法参与
- **永久封禁** — 严重或重复违规

## 适用范围

本准则适用于：
- GitHub Issues 和 Pull Requests
- GitHub Discussions
- 与项目相关的所有讨论
